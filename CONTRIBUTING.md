# Contributing to AIOps Healing Platform

Thank you for your interest in contributing!

## Pull Request Process

1.  Create a new branch for your feature or fix.
2.  Commit your changes with clear messages.
3.  Open a Pull Request against the `main` branch.
4.  Wait for **CodeRabbit** to review your changes.
5.  Address any feedback and merge once approved.

## Code Style

- **Python**: Follow PEP 8 (use `flake8` or `ruff`).
- **TypeScript/React**: Use ESLint and Prettier.
- **Terraform**: Run `terraform fmt` before committing.

## Reporting Issues

Please use the GitHub Issues tab to report bugs or request features.
